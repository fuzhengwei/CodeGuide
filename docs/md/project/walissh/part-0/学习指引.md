## 一、如何开始

### 1. 环境准备

首先，你需要在，星球「码农会锁」课程入口 -> 编程环境 -> [软件下载](https://t.zsxq.com/19Rnk98M0) 中，下载 IntelliJ IDEA、JDK17、Maven 3.8.x、MySQL 8.x（可以用docker安装）、WebStorm、[Nodejs 20+](https://nodejs.org/) 。如果你已经安装过，也可以不用重复安装。

另外，你还需要安装 Git 环境，用于拉取代码。Windows 可以在 [软件下载](https://t.zsxq.com/19Rnk98M0)  中获取安装包，Mac 电脑需要通过命令安装 `brew install git` 官网地址：[https://git-scm.com/install/mac](https://git-scm.com/install/mac)

>**Git 使用教程**（必看）：[https://bugstack.cn/md/road-map/git.html](https://bugstack.cn/md/road-map/git.html)

### 2. 项目下载

- 后端（walissh-server）：https://gitcode.net/KnowledgePlanet/walissh-server
- 前端（walissh-client）：https://gitcode.net/KnowledgePlanet/walissh-client

注意，不要直接点击 zip 包下载，正确✅的方式是 git clone 命令下载代码

- `git clone https://gitcode.net/KnowledgePlanet/walissh-server.git`
- `git clone https://gitcode.net/KnowledgePlanet/walissh-client.git`

> 项目下载完成后，你就可以通过 IntelliJ IDEA 打开项目了。这部分内容会在本节课程的视频演示中操作。
