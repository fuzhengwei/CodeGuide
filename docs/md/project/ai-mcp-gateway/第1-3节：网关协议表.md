## 一、本章诉求

设计 AI MCP Gateway 网关服务，所需的核心必备的数据库表。以用于存储，mcp 到 http 的协议转换处理，如，用户通过 sse 请求指定的网关 ID 对应的服务，则可以调用到对应的 http 服务接口。
