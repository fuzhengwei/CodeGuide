## 二、AI Shell 连接云服务器

### 1. 一键连接

AI Shell 智能识别服务器信息，告别手动输入 IP 和密码，自然语言描述即可连接。零配置，极速连接。

### 2. 自然语言操控

用中文描述你要做的事，AI 自动生成并执行命令，无需记忆复杂语法。AI 驱动，中文交互。

### 3. 安全可靠

执行前确认机制，危险操作自动拦截，生产环境操作无忧。安全拦截，操作确认。

## 三、AI Shell 操作流程

整体操作流程如下：

```
🖥️ AI Shell 终端（自然语言输入）
        ↓ 智能解析
🧠 AI Agent（意图识别 + 命令生成）
        ↓ SSH 执行
☁️ 云服务器（远程命令执行）
```

### 1. 连接步骤

- 打开 WaLiCode AI Shell
- 输入云服务器 IP / 域名
- 配置 SSH 密钥 / 密码
- AI 验证连接并进入交互

### 2. 示例对话

```bash
# 自然语言连接
连接到 47.98.123.45 服务器
查看 服务器系统版本
检查 磁盘使用情况
```
